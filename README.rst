2 - Installation
------------------

2.1 - Requirements 
-------------------

Required packages:

- boto ( at least version 2.8)
- prettytable
- simplejson
- argparse
- sqlite3
- isodate
- numpy
- cplex


2.2 - Manual Installation
--------------------------

Install from source:

::

	$ git clone https://github.com/sumo-tool/sumo.git
	$ cd sumo
	$ python setup.py install
