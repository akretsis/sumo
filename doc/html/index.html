SuMo: Smart Cloud Monitoring toolkit¶

Smart Cloud Monitoring toolkit, has been developed to contain the necessary mechanisms for collecting monitoring data from Amazon Web Services (AWS) and analyzing them.

SuMo makes easy for anyone, a researcher or an administrator, to monitor the owned instances, run the proposed mechanisms [] or implement new more intelligent ones. SuMo is open-source and available through github SuMo-tool: https://github.com/SuMo-tool. To the best of our knowledge it is the only open-source cloud monitoring toolkit for public clouds and in particular for AWS.

Sumo: Architecture¶

SuMo is composed of three main components/modules: cloudData, cloudKeeping and cloudForce. cloudData is responsible for collecting monitoring data, cloudKeeping contains a set of key performance indicators (KPI), while cloudForce incorporates a set of analytic and optimization algorithms.
