SuMo is a tool for smart public cloud monitoring and analysis


2 Installation 
=========================

2.1 Requirements
=========================

Required packages:

- boto
- cplex
- prettytable
- simplejson
- argparse
- sqlite3


2.2 Manual Installation 
==========================

Install from source:

$ git clone https://github.com/sumo-tool/sumo.git

$ cd sumo

$ python setup.py install
